# 激活教程 17.3版本
## 1.下载并安装navicat，可从官网下载(https://www.navicat.com.cn/products) 或 运行navicat17_premium_cs_x64.exe 
## 2.安装完成后不要运行直接关闭
## 3.解压 x64_Patch.zip (64位)，解压完成后，将解压后的文件 winmm.dll 放到 navicat安装路径下：
![路径](https://github.com/anxiulei/navicat17_PJ/blob/master/image/1.png?raw=true)
## 4.解压完成后打开navicat，成功解锁
![路径](https://github.com/anxiulei/navicat17_PJ/blob/master/image/navicat%EF%BC%89PJ.png?raw=true)
# 由于现在很多公司要求国产化数据库达梦，以下是配置navicat支持达梦数据库的驱动配置，若不需要达梦则忽略
## 1.官网下载达梦驱动或（dameng_odbc_win.zip）解压 ，解压完成后，打开 dameng_odbc_win文件夹，管理员身份运行 install_odbc.bat 文件
## 2.然后到电脑的环境变量中进行配置达梦数据库的驱动。path路径为达梦驱动（dameng_odbc_win）的bin路径
![路径](https://github.com/anxiulei/navicat17_PJ/blob/master/image/2.png?raw=true)
## 3.此时可以连接达梦了
![路径](https://github.com/anxiulei/navicat17_PJ/blob/master/image/DM.png?raw=true)
